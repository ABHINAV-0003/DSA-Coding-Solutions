## Detailed Descriptions

### 1. 2sum
**File:** `2sum`  
**Description:** Finds two numbers in an array that sum up to a target value.

### 2. Sort Array 0, 1, 2
**File:** `Sort_Array_0_1_2`  
**Description:** Sorts an array of 0s, 1s, and 2s.

### 3. Majority Element
**File:** `Majority_Element`  
**Description:** Finds the majority element in an array.

### 4. Maximum Subarray
**File:** `Maximum_Subarray`  
**Description:** Finds the contiguous subarray with the maximum sum.

### 5. Best Time to Buy and Sell Stock
**File:** `Best_Time_Buy_Sell`  
**Description:** Determines the best time to buy and sell stock for maximum profit.

### 6. Rearrange Array Elements by Sign
**File:** `Rearrange_Array_Elements_by_Sign`  
**Description:** Rearranges the elements of an array based on their signs.

### 7. Longest Successive Elements
**File:** `Longest_Successive_Elements`  
**Description:** Finds the length of the longest consecutive elements sequence in an unsorted array.

### 8. Set Matrix Zero
**File:** `Set_Matrix_Zero`  
**Description:** Sets entire rows and columns to zero if an element in that row or column is zero.

### 9. Rotate Image
**File:** `Rotate_Image`  
**Description:** Rotates a given n x n 2D matrix by 90 degrees clockwise.

### 10. Spiral Order
**File:** `Spiral_Order`  
**Description:** Returns all elements of a matrix in spiral order.

### 11. Subarray Sum Equals K
**File:** `Subarray_Sum_Equals_K`  
**Description:** Finds the total number of continuous subarrays whose sum equals a given value k.

### 12. Majority Element N%3
**File:** `Majority_Element_N%3`  
**Description:** Finds all elements in an array that appear more than n/3 times.

### 13. 3SUM
**File:** `3_SUM`  
**Description:** Finds all unique triplets in the array which give the sum of zero.

### 14. 4SUM
**File:** `4_SUM`  
**Description:** Finds all unique quadruplets in the array which give the sum of the target value.

### 15. Maximum Subarray Sum Zero
**File:** `Max_Subarray_Sum_Zero`  
**Description:** Finds the length of the longest subarray with a sum of zero.

### 16. Subarray XOR Sum K
**File:** `Subarray_XOR_Sum_K`  
**Description:** Finds the total number of subarrays with a given XOR sum equal to k.

### 17. Merge Intervals
**File:** `Merge_Subinterval`  
**Description:** Merges all overlapping intervals in a list of intervals.

### 18. Missing and Repeating Elements
**File:** `Missing_Repeating`  
**Description:** Finds the missing and repeating elements in an array of size n.

### 19. Inversion Count
**File:** `Inversion_Count`  
**Description:** Counts the number of inversions in an array.

### 20. Reverse Pairs
**File:** `Reverse_Pairs`  
**Description:** Counts the number of reverse pairs in an array.

### 21. Maximum Product Subarray
**File:** `Max_Product_Subarray`  
**Description:** Finds the maximum product of any contiguous subarray.

### 22. Binary Search
**File:** `Binary_Search`  
**Description:** Implements binary search to find the position of a target value within a sorted array.

### 23. Lower Bound
**File:** `Lower_Bound`  
**Description:** Finds the first position in a sorted array where the target can be inserted without violating the order.

### 24. Upper Bound
**File:** `Upper_Bound`  
**Description:** Finds the last position in a sorted array where the target can be inserted without violating the order.

### 25. Search Insert Position
**File:** `Search_Insert_Position`  
**Description:** Finds the index at which a target should be inserted in a sorted array.

### 26. Floor and Ceil in Sorted Array
**File:** `Floor_Ceil_Sorted`  
**Description:** Finds the floor and ceil of a target value in a sorted array.

### 27. Floor and Ceil in Unsorted Array
**File:** `Floor_Ceil_Unsorted`  
**Description:** Finds the floor and ceil of a target value in an unsorted array.

### 28. First and Last Position
**File:** `First_and_Last_Position`  
**Description:** Finds the first and last position of a target value in a sorted array.

### 29. Count Occurrences
**File:** `Count_Occurrences`  
**Description:** Counts the number of occurrences of a target value in a sorted array.

### 30. Search in Rotated Sorted Array
**File:** `Search_Rotated_Sorted_Array`  
**Description:** Searches for a target value in a rotated sorted array.

### 31. Minimum in Rotated Sorted Array
**File:** `Minimum_Rotaed_Sorted_Array`  
**Description:** Finds the minimum element in a rotated sorted array.

### 32. Single Element in Sorted Array
**File:** `Single_Element_In_Sorted_Array`  
**Description:** Finds the single element that appears only once in a sorted array where every other element appears twice.

### 33. Koko Eating Bananas
**File:** `Koko_banana`  
**Description:** Determines the minimum integer K such that Koko can eat all the bananas within a given number of hours.

### 34. Flowers Bloom
**File:** `Flowers_Bloom`  
**Description:** Finds the number of days after which all flowers have bloomed.

### 35. Smallest Divisor
**File:** `Smallest_Divisor`  
**Description:** Finds the smallest divisor of a given number.

### 36. Ship Capacity
**File:** `Ship_Capacity`  
**Description:** Determines the minimum capacity of a ship to ship packages within D days.

### 37. Kth Missing Positive Number
**File:** `Kth_missing`  
**Description:** Finds the Kth missing positive number in a sorted array.

### 38. Aggressive Cows
**File:** `Aggresive_cows`  
**Description:** Determines the largest minimum distance between aggressive cows placed in stalls.
